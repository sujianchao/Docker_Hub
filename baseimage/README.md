##简介
官方下载的ubuntu镜像的软件源以及内部的软件都没有更新。所以我利用了官方的ubuntu基础镜像`构建`了适合自己的基础镜像。它将作为以后自己和[多火工作室](duohuo.org)建立自己运行和测试容器等操作的`基础镜像`

它有哪些改进的地方？
* 更改中国科学与技术大学的软件源
* 安装了`vim`
* 修改了`vim`的配置文件，[连接见这里](https://github.com/adolphlwq/MyTools/tree/master/ubuntu)
